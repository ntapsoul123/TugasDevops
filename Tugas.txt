Tugas Minggu 5

Buat akun Git (untuk yang belum punya. boleh Github, boleh Gitlab, boleh Bitbucket, boleh git pribadi)
Buat sebuah file (boleh txt, boleh source code)
Commit file tersebut ke repository
Edit filenya (ditambahkan atau dikurangi)
Commit file yang sudah diubah
Buat branch baru
Edit file di branch
Commit hasil edit file di branch
Merge branch ke master
Commit file di master

Yang dikumpulkan link repo git public.
